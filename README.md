NexusNews merupakan project website portal berita/artikel berita yang memiliki fitur diantaranya :

🎯 FITUR UTAMA
Login & Role (admin / editor),
CRUD Artikel (tambah, edit, hapus).

📂 Manajemen Kategori,
Manajemen User + reset password.

📊 ANALYTICS
Views per artikel,
Trending harian,
Grafik views (Chart.js),
Top artikel (berdasarkan views)/

🎨 FRONTEND
Hero carousel (slider berita),
List artikel + detail page,
Search & pagination (DataTables).

⚡ UI/UX
Dark mode modern,
Responsive (Bootstrap),
Badge otomatis (Viral / Trending / Baru).
